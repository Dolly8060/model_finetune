# 模型评估报告

## 评估指标说明

- **BLEU**: 机器翻译标准评估指标，衡量n-gram精确度
- **ROUGE-1/2/L**: 文本摘要评估指标，衡量n-gram召回率
- **BERTScore**: 基于BERT的语义相似度评估

## 评估结果对比

| 指标 | 基础模型 | 微调模型 | 提升 |
|------|----------|----------|------|
| BLEU | 41.45 | 60.67 | +19.22 |
| ROUGE-1 | 48.15 | 72.50 | +24.35 |
| ROUGE-2 | 31.03 | 53.68 | +22.65 |
| ROUGE-L | 45.74 | 70.12 | +24.38 |
| BERTScore-P | 75.11 | 87.15 | +12.04 |
| BERTScore-R | 78.97 | 86.43 | +7.46 |
| BERTScore-F1 | 76.81 | 86.77 | +9.96 |

## 结论

微调后模型在以下指标上有提升: BLEU (+19.22), ROUGE-1 (+24.35), ROUGE-2 (+22.65), ROUGE-L (+24.38), BERTScore-P (+12.04), BERTScore-R (+7.46), BERTScore-F1 (+9.96)

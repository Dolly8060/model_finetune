# 模型评估报告

## 评估指标说明

- **BLEU**: 机器翻译标准评估指标，衡量n-gram精确度
- **ROUGE-1/2/L**: 文本摘要评估指标，衡量n-gram召回率
- **BERTScore**: 基于BERT的语义相似度评估

## 评估结果对比

| 指标 | 基础模型 | 微调模型 | 提升 |
|------|----------|----------|------|
| BLEU | 61.27 | 75.16 | +13.88 |
| ROUGE-1 | 48.03 | 61.57 | +13.55 |
| ROUGE-2 | 34.71 | 45.17 | +10.45 |
| ROUGE-L | 45.52 | 59.57 | +14.05 |
| BERTScore-P | 74.52 | 86.41 | +11.90 |
| BERTScore-R | 78.90 | 85.74 | +6.84 |
| BERTScore-F1 | 76.45 | 86.06 | +9.61 |

## 结论

微调后模型在以下指标上有提升: BLEU (+13.88), ROUGE-1 (+13.55), ROUGE-2 (+10.45), ROUGE-L (+14.05), BERTScore-P (+11.90), BERTScore-R (+6.84), BERTScore-F1 (+9.61)

# 模型评估报告

## 评估指标说明

- **BLEU**: 机器翻译标准评估指标，主要用于翻译子集
- **ROUGE-1/2/L**: 文本摘要评估指标，主要用于总结子集
- **BERTScore**: 基于BERT的语义相似度评估

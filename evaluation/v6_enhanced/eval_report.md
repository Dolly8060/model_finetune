# 模型评估报告

## 评估指标说明

### 翻译/总结任务指标
- **BLEU**: 机器翻译标准评估指标，衡量n-gram重合度
- **ROUGE-1/2/L**: 文本摘要评估指标，衡量词/短语覆盖率
- **BERTScore**: 基于BERT的语义相似度评估

### 指令遵循任务指标（核心）
- **IFR (Instruction Following Rate)**: 约束级通过率，所有约束中被满足的比例
- **Strict Accuracy**: 样本级完全通过率，所有约束都满足的样本比例
- **Loose Accuracy**: 宽松通过率，至少满足一半约束的样本比例
- **约束类型分解**: 按字数/格式/关键词/结构等约束类型分别统计通过率
